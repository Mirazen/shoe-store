# Интернет-магазин обуви

Учебный проект интернет-магазина на Django с системой корзины, управлением остатками по размерам и аутентификацией

## Архитектура приложения

### Приложение `products`

#### Модели:
- `Product` - основная информация о товаре (название, бренд, цена, изображение)
- `Size` - справочник размеров
- `ProductSize` - связующая модель между товаром и размером, содержит информацию об остатках

#### Формы:
- `AddToCartForm` - форма для добавления товара в корзину с выбором размера и количества

#### Представления:
- `add_to_cart` - добавление товара в корзину
- `catalog` - отображение каталога товаров

#### Шаблоны:
- `add_to_cart.html` - страница выбора размера и количества для добавления в корзину
- `catalog.html` - страница каталога товаров

### Приложение `cart`
Содержит модели и логику для работы с корзиной:

#### Модели:
- `Cart` - корзина пользователя
- `CartItem` - элемент корзины (ссылка на корзину, размер товара и количество)

#### Представления:
- `view_cart` - отображение содержимого корзины
- `remove_from_cart` - удаление товара из корзины

#### Шаблоны:
- `cart.html` - страница корзины

### Приложение `accounts`
Содержит логику аутентификации и профиля пользователя:

#### Представления:
- `profile` - страница профиля пользователя

#### Шаблоны:
- `profile.html` - страница профиля пользователя

## Технологический стек
- **Python**: `>=3.11`
- **Framework**: Django 5.2.10
- **Images**: Pillow
- **Linting**: Ruff
- **Package Manager**: uv (рекомендуется)


## Установка и запуск

### Способ 1: через uv
1. Клонируйте репозиторий
```bash
git clone https://github.com/Mirazen/shoe-store.git
cd shoe-store
```

2. Установите зависимости (создаст .venv автоматически)
```bash
uv sync
```

3. Выполните миграции и создайте админа
```bash
uv run manage.py migrate
uv run manage.py createsuperuser
```

4. Запустите сервер
```bash
uv run manage.py runserver
```

### Способ 2: через pip
1. Клонируйте репозиторий
```bash
git clone https://github.com/Mirazen/shoe-store.git
cd shoe-store
```

2. Создайте виртуальное окружение
```bash
python -m venv venv
```

3. Активируйте его
```bash
# Linux/Mac:
source venv/bin/activate
# Windows:
venv\Scripts\activate
```

4. Установите зависимости
```bash
pip install -r requirements.txt
```

5. Выполните миграции и создайте админа
```bash
python manage.py migrate
python manage.py createsuperuser
```

6. Запустите сервер
```bash
python manage.py runserver
```
## Особенности реализации
- Хранение корзины в базе данных
- Аутентификация пользователей с использованием встроенных средств Django
- Поддержка различных размеров для товаров
- Фильтрация доступных размеров при добавлении товара в корзину
